This project is to test out the Swift 3.0.1 functionality, and it does the following:
    - Creation/editing/removal of files
    - iOS file browsing
    - file encryption
    - dropbox, OneDrive, google drive, mega, and iCloud file syncing options
    - Image viewing
    - Video playback
    - Audio playback
    - File downloading
    - Data compression
